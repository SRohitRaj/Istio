The cloudwatch adapter enables Istio to deliver metrics to Amazon CloudWatch.

Params

Configuration for the cloudwatch adapter.

Field Type Description
namespace string

CloudWatch metric namespace.
metricInfo map<string, Params.MetricDatum>

A map of Istio metric name to CloudWatch metric info.

Params.MetricDatum

CloudWatch metric format.

Field Type Description
storageResolution int64

The second resolution of the metric. Valid values are 1 and 60. https://docs.aws.amazon.com/AmazonCloudWatch/latest/APIReference/API_MetricDatum.html
unit string

The unit of the metric. Must be valid cloudwatch unit value. https://docs.aws.amazon.com/AmazonCloudWatch/latest/APIReference/API_MetricDatum.html
